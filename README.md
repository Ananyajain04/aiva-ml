# aiva-ml

This is a small OCR project using EasyOCR.


## What is inside

- `orc/easy_ocr.ipynb`: main notebook
- `orc/optimized_output/`: saved optimized images
- `requirements.txt`: python packages

## How to run

1. Create and activate a virtual environment.
2. Install requirements:
   `pip install -r requirements.txt`
3. Open and run `orc/easy_ocr.ipynb`.

## Notes

- Put your input image in the expected location and update `IMAGE_PATH` in the notebook if needed.
- Output files are written to `orc/optimized_output/`.
